## 📝 Module Summaries

### Task 1 – Digital Literacy Awareness Infographic (Module 1)
Created a single-page infographic using **Canva** covering three core topics: what digital literacy means, safe internet practices, and building a professional online presence. The infographic is designed for fellow first-year students to quickly grasp essential digital skills.

### Task 2 – Student Digital Portfolio (Module 2)
Set up accounts on **GitHub**, **LinkedIn**, and **Kaggle**. Each platform serves a specific purpose: GitHub for code repositories and projects, LinkedIn for professional networking, and Kaggle for data science practice and competitions. A profile README was created on GitHub.

### Task 3 – Coding & Collaboration Platforms (Module 3)
Completed a beginner challenge on **HackerRank** (Solve Me First) to get comfortable with competitive programming. Also created a **Google Form** with 5 questions as a Digital Literacy Awareness Quiz for batchmates.

🔗 **Digital Literacy Quiz (Google Form):**(https://docs.google.com/forms/d/e/1FAIpQLScWvwziVOERgcG3d5F2holJbfi6RHY84XtGeHjTkASJyhuOOA/viewform?usp=publish-editor)

### Task 4 – Professional Email & Etiquette Guide (Module 4)
Drafted two professional emails — one requesting an assignment extension from a professor, and one expressing interest in a summer internship. Also created a Social Media Do's and Don'ts checklist for college students.

### Task 5 – Cybercrime Awareness: Cyberbullying (Module 5)
Researched **Cyberbullying** as a major online threat affecting college students. Wrote a detailed case study describing how it occurs, who is targeted, and its consequences. Prepared an 8-point prevention checklist with specific guidance for Indian college students, including reporting channels: **cybercrime.gov.in** and Helpline **1930**.
